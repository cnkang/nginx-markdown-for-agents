# 0.7.0 Validation Matrix

This matrix documents the validation checks that constitute the 0.7.0
release gate. Each row maps a gate area to the verification commands and
artifacts that the gate validator checks for.

## Gate 1: Build and Header Integrity

| Check | Command | Notes |
|-------|---------|-------|
| FFI ABI drift check | `make build && make check-headers` | C/Rust header sync |

## Gate 2: FFI Boundary Tests

| Check | Command | Notes |
|-------|---------|-------|
| C unit tests | `make test-nginx-unit` | State machine + config parsing |
| Rust unit tests | `make test-rust` | Converter + FFI boundary |
| Layout tests | Layout tests | Struct layout assertions |
