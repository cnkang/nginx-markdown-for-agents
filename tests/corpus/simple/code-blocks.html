Code Block Examples

Simple Code Block

function hello() {
    console.log("Hello, World!");
}

Code Block with Language

def greet(name):
    print(f"Hello, {name}!")

greet("World")

Code Block with JavaScript

const add = (a, b) => a + b;
console.log(add(2, 3));

Inline Code vs Block

Use console.log() for debugging.

Here's a full example:

const debug = (msg) => {
    console.log(`[DEBUG] ${msg}`);
};

Multiple Code Blocks

First example:

echo "Hello"

Second example:

echo "World"
